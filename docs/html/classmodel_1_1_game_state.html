Description détaillée

Représente l'état global d'une partie d'Othello/Reversi.

Cette classe gère le plateau, les deux joueurs, le joueur courant, le statut de fin de partie, le gagnant éventuel ainsi que l'historique des coups joués.

Documentation des constructeurs et destructeur

◆ GameState() [1/2]

model.GameState.GameState ( int boardSize)
inline

Construit un nouvel état de jeu avec un plateau de taille donnée.

Initialise le plateau, crée les deux joueurs (noir et blanc), définit le joueur courant (noir commence), marque la partie comme non terminée et prépare l'historique des coups.

Paramètres
boardSizeTaille du plateau (nombre de cases par côté).

◆ GameState() [2/2]

model.GameState.GameState ( )
inline

Construit un nouvel état de jeu avec un plateau de taille par défaut.

Utilise la taille standard de 8 cases par côté pour le plateau. Ce constructeur est fourni pour la compatibilité avec un usage classique.

◆ setCurrentPlayer()

void model.GameState.setCurrentPlayer ( Cell.CellState color)
inline

Définit le joueur courant à partir d'une couleur.

Si la couleur passée correspond à celle de player1, celui-ci devient le joueur courant ; sinon, c'est player2 qui devient le joueur courant.

Paramètres
colorCouleur du joueur à définir comme joueur courant (généralement Cell.CellState#BLACK ou Cell.CellState#WHITE).

◆ switchPlayer()

void model.GameState.switchPlayer ( )
inline

Change le joueur courant pour l'autre joueur.

Si le joueur courant est player1, il devient player2, et inversement. Cette méthode est typiquement appelée après qu'un coup valide a été joué.
