Description détaillée

Représente un coup joué dans une partie d'Othello/Reversi.

Un coup est défini par la position ciblée sur le plateau (coordonnées x, y), le joueur qui effectue ce coup, ainsi que la liste des pions capturés (retournés) à la suite de ce coup.
