Description détaillée

Représente la fenêtre principale de l'application Othello / Reversi.

Cette classe hérite de JFrame et constitue le point d'entrée graphique de l'application. Elle gère l'affichage du menu principal, l'initialisation de la fenêtre ainsi que le lancement d'une nouvelle partie via le contrôleur du jeu.

Documentation des constructeurs et destructeur

◆ OthelloApp()

view.OthelloApp.OthelloApp ( )
inline

Construit la fenêtre principale de l'application.

Ce constructeur initialise les propriétés générales de la fenêtre : titre, comportement à la fermeture, dimensions et position à l'écran. Il affiche ensuite le menu principal de l'application.

Documentation des fonctions membres

◆ main()

static void view.OthelloApp.main ( String[] args)
inlinestatic

Point d'entrée principal de l'application.

Cette méthode démarre l'interface graphique sur le thread événementiel de Swing à l'aide de SwingUtilities#invokeLater(Runnable), puis crée et affiche la fenêtre principale de l'application.

Paramètres
argsArguments de la ligne de commande transmis au programme. Ils ne sont pas utilisés dans cette application.

◆ showMenu()

void view.OthelloApp.showMenu ( )
inline

Affiche le menu principal de l'application.

Cette méthode vide le contenu actuel de la fenêtre, crée un panneau de menu vertical, ajoute le titre et les boutons d'action disponibles (joueur contre joueur, chargement de partie, quitter), puis rafraîchit l'affichage de la fenêtre.

◆ showMenuFromGame()

void view.OthelloApp.showMenuFromGame ( )
inline

Réaffiche le menu principal depuis l'écran de jeu.

Cette méthode peut être appelée lorsqu'une partie se termine ou lorsqu'un retour au menu principal est demandé depuis l'interface du jeu.

◆ startGame()

void view.OthelloApp.startGame ( int mode)
inlineprivate

Lance une nouvelle partie selon le mode choisi.

Cette méthode crée une nouvelle instance du contrôleur du jeu, associée à cette fenêtre, puis demande au contrôleur de démarrer la partie selon le mode sélectionné.

Paramètres
modeMode de jeu à lancer. Par exemple :
  • 1 pour joueur contre joueur ;
  • 3 pour charger une partie.
