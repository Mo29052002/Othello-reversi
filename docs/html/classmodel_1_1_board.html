Description détaillée

Représente le plateau de jeu d'Othello/Reversi.

Cette classe gère la grille de cellules ainsi que l'état global du plateau (taille, disposition initiale des pions et opérations de base sur les cases).

◆ initializeBoard()

void model.Board.initializeBoard ( )
inlineprivate

Initialise la grille de cellules du plateau.

Crée chaque cellule avec ses coordonnées (x, y) et positionne ensuite les quatre pions de départ au centre du plateau :

  • deux pions blancs en diagonale
  • deux pions noirs en diagonale conformément aux règles classiques d'Othello.

◆ isFull()

boolean model.Board.isFull ( )
inline

Indique si le plateau est entièrement rempli.

Le plateau est considéré comme plein lorsqu'aucune cellule n'est vide, c'est-à-dire qu'il n'existe plus de case disponible pour poser un pion.

Renvoie
true si toutes les cellules contiennent un pion, false dès qu'au moins une cellule est encore vide.

inline

Place un pion sur la case indiquée si la position est valide.

La méthode modifie l'état de la cellule aux coordonnées (x, y) en lui attribuant l'état spécifié (par exemple NOIR ou BLANC). Si la position est invalide, aucun changement n'est effectué.

Paramètres
xCoordonnée horizontale de la case où placer le pion.
yCoordonnée verticale de la case où placer le pion.
stateNouvel état de la cellule, représentant la couleur du pion (par exemple Cell.CellState#BLACK ou Cell.CellState#WHITE).
