\doxysubsection{Description détaillée}
Représente le plateau de jeu d\textquotesingle{}Othello/\+Reversi. 

Cette classe gère la grille de cellules ainsi que l\textquotesingle{}état global du plateau (taille, disposition initiale des pions et opérations de base sur les cases).
