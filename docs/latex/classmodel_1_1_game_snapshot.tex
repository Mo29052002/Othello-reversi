\doxysubsection{Description détaillée}
Représente un instantané de l\textquotesingle{}état d\textquotesingle{}une partie d\textquotesingle{}Othello/\+Reversi. 

Cette classe capture toutes les informations nécessaires pour décrire un état précis de la partie \+: le coup joué pour atteindre cet état, l\textquotesingle{}état du plateau, le joueur dont c\textquotesingle{}est le tour, les scores et le numéro du coup. Elle peut être utilisée pour l\textquotesingle{}historique de la partie, l\textquotesingle{}annulation de coups, ou la sauvegarde/restauration de l\textquotesingle{}état du jeu.
