\doxysubsection{Description détaillée}
Représente l\textquotesingle{}état global d\textquotesingle{}une partie d\textquotesingle{}Othello/\+Reversi. 

Cette classe gère le plateau, les deux joueurs, le joueur courant, le statut de fin de partie, le gagnant éventuel ainsi que l\textquotesingle{}historique des coups joués.
